Introduccion

La configuración de un intérprete de comandos consiste mayormente en establecer las variables de entorno, los aliases de comandos y el formato del prompt que se necesiten. Dependiendo del tipo de uso que se le dará al intérprete de comandos, habrá diferentes necesidades de configuración de estos valores.

Existen dos modalidades de uso del intérprete de comandos bash:

interactiva
