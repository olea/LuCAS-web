Manejo de archivos

Desde los dos cuadros de diálogo es posible acceder a una serie de funciones sencillas para crear carpetas, copiar y borrar archivos, etc. Aunque esto es mucho más cómodo de realizar desde el Administrador de archivos de KDE, algunas veces es útil disponer de estas opciones directamente desde StarOffice. Si se pulsa sobre el nombre de un documento con el botón derecho del mouse, aparece un menú de contexto que permite algunas manipulaciones.
