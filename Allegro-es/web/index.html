Allegro-es

Bienvenido a Allegro-es, el proyecto de traducción de la documentación y demás ficheros de interés que vienen con Allegro, una librería de videojuegos distribuída líbremente y que funciona en las siguientes plataformas: DOS, Unix (Linux, FreeBSD, Irix, Solaris), Windows, y BeOS (las versiones MacOS y QNX están en estado alpha). Tiene muchas funciones de gráficos, sonidos, entrada del usuario (teclado, ratón y joystick) y temporizadores. También tiene funciones matemáticas en punto fijo y coma flotante, funciones 3d, funciones para manejar ficheros, ficheros de datos comprimidos y una interfáz gráfica.

Allegro 3.11

Si desea obtener la documentación de Allegro 3.11, puede bajarse el fichero spdocs311.zip (164_KB) desde el servidor de LuCAS, o si lo prefiere, puede bajarse el fichero desde el servidor de ftp ftp.simtel.net o cualquiera de sus mirrors.

Recuerde leer antes las instrucciones de instalación, que por supuesto están incluídas en el fichero comprimido en el archivo de texto leame.txt.

Si desea ver la versión html de la documentación 3.11, siga este enlace.

Allegro 4.0.1

La versión 3.11 de Allegro es exclusiva para sistemas DOS. Las versiones posteriores estan portadas a varias plataformas, como Windows o Linux. Su documentación se genera mediante un programa C propio de Allegro (incluído en el archivo), con salidas en formato txt, html, rtf o info. Esta versión está traducida hasta la versión 4.0.1 del día 15 de Diciembre del 2001. Todavía no está totalmente finalizada y quedan algunos cabos sueltos por traducir, pero se puede decir que es casi usable. Es por esta razón que todavía no se distribuye cono documentación definitiva, sino como alpha. Aquí hay una lista de las cosas que faltan por hacer.

Para generar la documentación descarge uno de los siguientes archivos de código fuente, preparados para DOS y Linux respectivamente , y compile la documentación con el comando make (necesitará disponer del compilador DJGPP o GCC respectivamente):

* allegres.zip - DOS, 306Kb.
* Allegro-es-4.0.1-alpha3.tar.gz - Linux, 277Kb.
* Allegro-es-4.0.1-alpha3.tar.bz2 - Linux, 217Kb.

Si desea ver la versión html de la documentación 4.0.1, siga este enlace.
