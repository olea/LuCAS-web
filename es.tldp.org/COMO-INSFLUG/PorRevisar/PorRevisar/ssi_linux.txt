  Configuración y uso de Server Side Includes (SSI) en Linux
  Paco Brufal, <pbrufal@linuxfreak.com>
  Versión 0.1, Octubre 1999

  Este pequeño documento explica de manera sencilla cómo configurar y
  usar las instrucciones Server Side Includes (SSI) para crear documen-
  tos HTML de forma dinámica y sencilla. Asi mismo, este documento se
  distribuye SIN NINGUNA GARANTIA bajo licencia GPL. No hace falta decir
  que se admiten sugerencias, ejemplos, correcciones, etc. para posteri-
  ores revisiones.
