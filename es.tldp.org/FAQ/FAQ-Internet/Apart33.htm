3.3.0.- W95/NT

Probablemente el sistema operativo más utilizado por el usuario de "a pie" para acceder a Internet y posiblemente también el más criticado ;-)

3.3.1.- Infovia

3.3.1.1.- Como se configura ACCESO TELEFONICO A REDES DE W95 por Infovia ?

Panel de control, Red:
  • Dirección IP: Obtener automático
  • Configuración WINS: Desactivar
  • Configuración DNS: Activar
  • DNS. Host: 10.0.1.1
  • Dominio: inf

    •  
    Creas una nueva conexión:
  • Telf: 055
  • Usuario: infovia
  • Clave : infovia

    •  
    En las propiedades de la conexión, Tipo de servidor:
  • Tipo de servidor: PPP: Windows 95, Windows NT 3.5, Internet
  • Activar: Activar compresión de software TCP/IP Todo lo demás desactivado.

    •  
    Configuarción TCP/IP:
  • Dirección IP asignada por el servidor
  • Direcciones del servidor asignadas por el usuario: 10.0.1.1
  • Los dos checks de abajo activados.

    •  
    Tienes que poner como página para empezar... http://10.0.1.1/editadas/Bienvenida.html

    3.3.2.- Ibertex

    3.3.2.1.- Se puede acceder a Ibertex a través de Infovía ?

    Si, pero sólo a los servidores gratuitos, del 031 y NO A TODOS; sólamente a los que están preparados para ello. A veces de algunos "no preparados" puedes acceder a la pantalla de entrada, pero eres expulsado inmediatamente.

    3.3.2.2.- Con que tarifa ?

    Con la misma que infovía en general, es decir: llamada metropolitana.

    3.3.2.3.- Con que software se puede acceder ?

    a) Precisas primero una conexión Infovía (propiamente de infovía, no la que tal vez uses para acceder a tu proveedor de Internet por Infovía). La conexión con Infovía la puedes establecer con cualquier software como el acceso telefónico a Redes del w95, el Trumpet para windows 3.1, etc..
    b) Pero ADEMAS necesitas un programa que te permita la conexión en Ibertex desde Infovía. Esto se puede conseguir:
    - Con el Tesawin, incluido en el CD de Infovía (gratuito pidiéndolo a Timofónica y distribuido también por muchas revistas). El Tesawin requiere que accedas primero con un navegador a la dirección (OJO, dirección de Infovía,no de Internet) de Gesvía y desde allí se ejecuta el Tesawin. La dirección de Gesvía debiera aparecer en el buscador de Infovía.
    - Al parecer, también sirven algunas versiones recientes del Amaris.

    3.3.2.4.- Dónde puedo conseguir más información ?

    En estos teléfonos no siempre das con la persona más informada o más dispuesta a informar, pero por intentarlo que no quede:
    900.500.055 (gratuito, Infovía) 902.230.240 (Gesvía)

    3.3.3.- Wingate

    3.3.3.1.- Qué es Wingate ?

    Wingate es un software (proxie/firewall) que permite conectar los ordenadores de una red local a Internet a traves una sola cuenta.
