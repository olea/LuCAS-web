3.2.0.- WIN 3.1x

3.2.1.- Acceso a InfoVía sin el software de conexión desarrollado por Telefónica

El programa de comunicaciones a utilizar es el Trumpet (TPCMAN.EXE), configurado del siguiente modo:

En la opción Setup del menú File colocar los siguientes valores:

  ip : 0.0.0.0
  netmask : 0.0.0.0 Default gateway : 0.0.0.0
  name Server : 10.0.1.1 Time Server    :
  Domain Suffix : inf
  Packet Vector : 00 MTU : 1006 TCP RWIN : 4096 TCP MSS : 966
  Demand load timeout (secs) : 5 TCP RTO MAX : 60

  internal SLIP[ ] internal PPP[X]
  Slip Port: 2   On line Status Detection
  Baud Rate: 57600  [x] None
  [x] Hardware Handshake  [ ] Dcd (rlsd) check
  [ ] Van Jacobson CSLIP comp. [ ] Dsr check

*Nota: Slip Port, es el puerto del modem, en el ejemplo es el com2. Baud Rate es la velocidad de transferencia con el módem, para módems de 28.800 ó 33.600, subir este valor hasta 115.000

Los valores en PPP Options, también del menú File son:

  Username: infovia
  Password: infovia

*Nota: Estos han de estar en minúsculas y sin acentos!!

Esto sólo es valido para InfoVía, para navegar por Inet el proveedor de acceso elegido nos debe dar los valores para los campos Name Sever y Domain Suffix del Setup. PPP Option debe quedar:

[x]  Use Password Authentication Protocol  (PAP)
Username: usuario@proveedor
Password:

*Nota: Ejemplo de username: pepe@arrakis, juan@bbv

Para tener disponibles varias conexiones, sin necesidad de modificar cada vez la configuración del Trumpet :
 

  • Ejecutar el Trumpet, configurarlo para una de las conexiones, al salir del programa se habrá actualizado el fichero de configuración llamado tumpwsk.ini, copiarlo con otro nombre y la misma extensión. Volver a repetir el proceso por cada una de las conexiones a definir.

    •  
  • Crear un icono por cada una de las conexiones, el la Línea de comando poner: C:\directorio\TCPMAN.EXE -inipath=c:\directorio\fichero.ini
*Nota:  C: es el disco. \directorio\  el subdirectorio donde está el programa o el fichero de configuración, fichero.ini es el nombre de cada uno de los ficheros de configuración creados anteriormente.
