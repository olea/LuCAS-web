Para activar SMP

Configure el kernel y conteste Y (sí) a CONFIG_SMP

Si está usando LILO, es útil disponer de ambas imágenes con y
sin SMP a mano. Edite /etc/lilo.conf para crear una nueva entrada
para otra imagen del núcleo llamada linux-smp o similar.

La próxima vez que compile el núcleo, mientras ejecuta un núcleo
con soporte SMP, edite el fichero linux/Makefile y cambien
"MAKE=make" por "MAKE=make -jN" (donde N= número de CPUs + 1,
o si tiene toneladas de memoria o swap simplemente use -j sin
especificar ningún número). Siéntase libre de experimentar con esto.

Por supuesto, debería medir cuánto tiempo lleva cada compilación :)
Ejemplo:
   make config
   time -v sh -c 'make dep ; make clean install modules modules_install'

Si está usando algunas máquinas Compaq que cumplen MP necesitará cambiar
el sistema operativo en las opciones de la BIOS a "Unixware" - (no me
pregunte por qué los ordenadores Compaq no funcionan sin ello).
