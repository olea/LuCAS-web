Descargas

Paquetes de documentación en castellano.

La labor de NuLies, al menos por el momento, se limita a la traducción al español del directorio Documentation/, y de los ficheros necesarios en otros directorios dentro de las fuentes del núcleo para conseguir tener traducidos los menús de configuración tanto de texto (make config), de ncurses (make menuconfig) o gráficos (make xconfig), y la ayuda de tales sistemas de configuración del núcleo.

También están dentro del proyecto cualesquiera otros documentos que vivan en el directorio Documentation/ y sus subdirectorios.

La forma de instalar los paquetes de NuLies es la siguiente (se *supone* que tiene permitido escribir en el directorio de las fuentes del núcleo):

  • Descargue el último paquete de NuLies correspondiente a su versión del núcleo. IMPORTANTE: Tan sólo es válido el paquete de *su versión*. Si instala un paquete posterior o anterior, el parche no se aplicará bien y puede que haya cosas que no estén donde o como debieran.

    "Último paquete" significa la última de las versiones disponibles para su versión del núcleo. Los paquetes de NuLies tienen un nombre del tipo:
        NuLies-version_núcleo_linux-versión_de_NuLies.gz.
    (Ej: NuLies-2.4.0-2.gz es la versión 2 del paquete NuLies para el núcleo 2.4.0).
  • Sitúese en el directorio donde tenga descomprimidas las fuentes del núleo (comúnmente /usr/src/linux/).
  • Copie en ese directorio el paquete de NuLies antes mencionado.
  • Ahora mismo debería estar en el directorio /usr/src/linux/ o similar. Ejecute la siguiente orden:
        zcat NuLies-x.y.z-t.gz | patch -p1
    donde x.y.z es la versión de su núcleo y t la versión del paquete NuLies.
  • La salida de esta orden será similar a: 
        patching file Documentation/00-INDEX
    patching file Documentation/Changes
    patching file Documentation/binfmt_misc.txt
    patching file Documentation/cdrom/00-INDEX
    patching file Documentation/fb/vesafb.txt
    patching file Documentation/kbuild/commands.txt
    patching file Documentation/logo.txt
    patching file Documentation/m68k/00-INDEX
    patching file Documentation/ramdisk.txt
    patching file Documentation/smp.txt
    patching file Documentation/sound/CS4232
    patching file Documentation/sound/MAD16
    patching file Documentation/sound/MultiSound
    patching file Documentation/sound/OPL3
    [...]
  • ¡Hecho! A partir de ahora puede disfrutar de todos los documentos que el proyecto NuLies pone a su disposición.

Si desea desinstalar el paquete NuLies, ya sea por alguna razón esotérica :o), o más típicamente porque desea aplicar un parche oficial al núcleo, la manera de proceder es la siguiente:

  • Sitúese en el directorio de las fuentes del núcleo (típicamente /usr/src/linux/).
  • Ejecute la siguiente orden (note que ahora hay un -R que antes no estaba): 
        zcat NuLies-x.y.z-t.gz | patch -p1 -R
    donde x.y.z es la versión de su núcleo y t la versión del paquete NuLies.
  • La salida de esta orden será similar a: 
        patching file Documentation/00-INDEX
    patching file Documentation/Changes
    patching file Documentation/binfmt_misc.txt
    patching file Documentation/cdrom/00-INDEX
    patching file Documentation/fb/vesafb.txt
    patching file Documentation/kbuild/commands.txt
    patching file Documentation/logo.txt
    patching file Documentation/m68k/00-INDEX
    patching file Documentation/ramdisk.txt
    patching file Documentation/smp.txt
    patching file Documentation/sound/CS4232
    patching file Documentation/sound/MAD16
    patching file Documentation/sound/MultiSound
    patching file Documentation/sound/OPL3
    [...]
  • Y el paquete NuLies desaparecerá como si nunca hubiera existido.

    Si tiene algún problema, no dude en escribirnos. ¡Feliz NuLies! :-)
