Proyecto LuCAS - Servicio de listas de correo-e

LuCAS, a través del servicio de listas de correo que puesto en marcha por HispaLinux, pone a disposición de la comunidad de desarrolladores de documentación listas de correo para la infraestructura de proyectos.

Este servicio cuenta con la ventaja de que las listas son archivadas automáticamente y accesibles por la web, además de que pueden ser fácilmente administradas desde un navegador. Todo ello gracias a un servidor mailman.

Para pedir la apertura de una lista dirígete a lucas-coord.

Lista LuCAS

Además está disponible la lista principal de consultas y coordinación. Para saber cómo participar o cómo suscribirse debe leer las normas de uso. Para examinar la lista desde la web visite el archivo html.

Lista LuCAS-desarrollo

En esta lista, se habla de todo lo referente a proyectos para mejorar a LuCAS, si quieres ayudarnos en esta tarea, únete a la lista. Para examinar la lista desde la web visite el archivo html.

Tambien visita la página web dedicada a esta importante parte de LuCAS.

Lista LuCAS-webmasters

Puedes encontrar todo lo referente a la web de LuCAS, en esta lista, siempre necesitamos gente con ideas nuevas, asi que únete a la lista. Para examinar la lista desde la web visite el archivo html.

Lista LuCAS-voluntarios

En esta lista, puedes ayudar en los diferentes proyectos o tareas de LuCAS, los coordinadores de proyectos, o de LuCAS solicitan voluntarios para ayudar en algunas tareas, únete a la lista. Para examinar la lista desde la web visite el archivo html.

Lista docbook-ayuda

En LuCAS estamos migrando nuestros documentos a Docbook y preferimos este formato, en esta lista puedes encontrar ayuda sobre Docbook, únete a la lista. Para examinar la lista desde la web visite el archivo html.
