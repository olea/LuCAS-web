Guía de Administración de Sistemas GNU/Linux

Una introducción a la administración de sistemas GNU/Linux para novatos.

Esta es la página de referencia de coordinación del proyecto de traducción del libro "The Linux System Administrator's Guide", versión 0.8
desarrollado por The Linux Documentation Project.

En este documento se describen los aspectos del uso de un sistema GNU/Linux relativos a la administración del sistema. Está destinado a personas con pocos conocimientos, pero que ya dominan al menos los conceptos básicos sobre la utilización normal del mismo. Este manual tampoco explica cómo instalar GNU/Linux; dicho tema está desarrollado en el documento "Instalación y Primeros Pasos".

Descargar el libro (Download)

Formatos disponibles : texto, PDF, HTML (tgz, múltiples páginas) y HTML (tgz, Archivo individual).
Si tiene problemas puede obtener el libro utilizando las copias alojadas en Ibiblio (tldp).

Código fuente: módulo "doc-guia-admon-sistemas" del servidor CVS cvs.hispalinux.es.

Si no dispone de una cuenta en el servidor CVS puede obtener el código fuente del libro utilizando el usuario "anoncvs":

export CVSROOT=:pserver:anoncvs@cvs.hispalinux.es:/lucas
cvs login (password: anoncvs)
cvs co doc-guia-admon-sistemas

Nota acerca de la traducción: El nombre de la traducción coincide con la versión de la guía elaborada por el proyecto Debian. Esto puede traer confusiones, debido a que la traducción que elaboraremos, corresponde a la guía publicada por "The Linux Documentation Project" y no a la desarrollada por el proyecto Debian. Si tienes preguntas al respecto, no dudes en consultar.

El proyecto está coordinado por Rafael Ignacio Zurita. Ponte en contacto con él para cualquier cosa que necesites.
