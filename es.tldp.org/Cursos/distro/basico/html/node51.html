StarDesktop: El Entorno de StarOffice

Figura 5.2: StarDesktop, el entorno de trabajo de StarOffice

Cuando arranca StarOffice, ocupa su propia ventana, que es una ventana de aplicación. Dentro de esta ventana se situarán las ventanas de documento, una para cada documento que se desee visualizar, además de una ventana para el escritorio propio de StarOffice (en inglés, desktop). La figura 5.2 muestra una pantalla típica en la que, además de la ventana principal, se ve el desktop.
