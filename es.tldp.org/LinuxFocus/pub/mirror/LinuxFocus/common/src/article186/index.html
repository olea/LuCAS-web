sled-0.4, syslog support

Marc Kirchner <kirchner(at)stud.fh-heilbronn.de> has designed a version that uses syslog to log startup, shutdown and errors in the system log (usually /var/log/messages) via the the syslog(2) interface. You find this version at:
http://stud.fh-heilbronn.de/~kirchner/sled/
