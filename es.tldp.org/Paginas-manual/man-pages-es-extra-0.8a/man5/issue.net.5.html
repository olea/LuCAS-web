NOMBRE

issue.net - fichero de identificación para las sesiones de telnet  

DESCRIPCIÓN

El fichero /etc/issue.net es un fichero de texto que contiene un mensaje o una identificación del sistema que será impreso justo antes de que aparezca el indicador de comandos (prompt) de una sesión de telnet. Puede contener varias secuencias %carácter Las siguientes secuencias son soportadas por telnetd:

%t
- muestra el tty actual
%h
- muestra el nombre del nodo (FQDN)
%D
- muestra el nombre de dominio NIS
%d
- muestra la hora y fecha actual
%s
- muestra el nombre del sistema operativo
%m
- muestra el tipo de máquina (hardware)
%r
- muestra el número de liberación (release) del sistema operativo
%v
- muestra la versión del sistema operativo
%%
- muestra el carácter '%'

 

FICHEROS
