DESCRIPCIÓN

El comando mrd se usa para borrar subdirectorios de MS-DOS. Su sintáxis es:

mrd [-v] directoriomsdos [directoriosmsdos... ]
 

Mrd borra un directorio de un sistema de ficheros MS-DOS. Si el directorio no existe o no está vacío se produce un error.
