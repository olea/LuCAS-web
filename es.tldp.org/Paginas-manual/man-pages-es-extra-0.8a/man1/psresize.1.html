NOMBRE

psresize - múltiples páginas por hoja  

SYNOPSIS

psresize

[ -wwidth ] [ -hheight ] [ -ppaper ] [ -Wwidth ] [ -Hheight ] [ -Ppaper ] [ -q ] [ infile [ outfile ] ]  

DESCRIPCIÓN

Psresize

reescala y centra un documento en un tamaño de papel diferente. El fichero PostScript de entrada debe seguir los Convenios de Estucturación de Documentos de Adobe (Document Structuring Conventions).

La opción -w especifica la anchura del papel, y la opción -h especifica la altura del papel, normalmente especificada en cm o in para convertir puntos PostScript (1/72 de una pulgada) a centímetros o pulgadas. La opción -p puede utilizarse como alternativa para indicar el tamaño del papel de salida como a3, a4, a5, b5, letter, legal, tabloid, statement, executive, folio, quarto o 10x14. El tamaño por defecto del papel de salida es a4.

La opción -W indica la anchura del papel de entrada, y la opción -H indica la altura del papel de entrada. La opción -P se puede usar como alternativa para especificar el tamaño del papel de entrada. El tamaño del papel de entrada por defecto es a4.

Psresize normalmente imprime los números de página de las páginas que resultan. La opción -q suprime esto.  

EJEMPLOS

El siguiente comando puede usarse para convertir un documento de tamaño de papel A4 a papel de tamaño carta (letter):

psresize -PA4 -pletter in.ps out.ps
