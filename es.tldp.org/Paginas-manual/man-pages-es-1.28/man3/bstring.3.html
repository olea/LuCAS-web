NOMBRE

bcmp, bcopy, bzero, memccpy, memchr, memcmp, memcpy, memfrob, memmem, memmove, memset - operaciones sobre cadenas de bytes  

SINOPSIS

#include <string.h>

int bcmp(const void *s1, const void *s2, int n);

void bcopy(const void *src, void *dest, int n);

void bzero(void *s, int n);

void *memccpy(void *dest, const void *src, int c, size_t n);

void *memchr(const void *s, int c, size_t n);

int memcmp(const void *s1, const void *s2, size_t n);

void *memcpy(void *dest, const void *src, size_t n);

void *memfrob(void *s, size_t n);

DESCRIPCIÓN

Las funciones de cadenas de bytes ejecutan operaciones sobre cadenas que no terminan en NULL. Vea las páginas de manual individuales para las descripciones de cada una de las funciones.
