NOMBRE

wmemcpy - copia un vector de caracteres anchos  

SINOPSIS

#include <wchar.h>

wchar_t *wmemcpy (wchar_t *dest, const wchar_t *src, size_t n);
 

DESCRIPCIÓN

La función wmemcpy es equivalente para caracteres anchos a la función memcpy. Copia n caracteres anchos del vector que comienza en src al vector que comienza en dest.

Los vectores no se pueden solapar. Use wmemmove(3) para copiar entre vectores que se solapan.

El programador debe asegurar que hay espacio para al menos n caracteres anchos en dest.  

VALOR DEVUELTO

wmemcpy

devuelve dest.  

CONFORME A

ISO/ANSI C, UNIX98  

VÉASE TAMBIÉN

memcpy

(3), wmemmove(3), wcscpy(3)
