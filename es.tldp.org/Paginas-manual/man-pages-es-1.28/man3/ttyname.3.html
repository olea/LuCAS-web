NOMBRE

ttyname - devuelve el nombre de un terminal  

SINOPSIS

#include <unistd.h>

char *ttyname ( int desc );
 

DESCRIPCIÓN

Devuelve un puntero al nombre del camino del terminal que se abre con el descriptor de fichero desc, o NULL si hay algún error (p.e.: si desc no está conectado a un terminal).  

CONFORME A

POSIX.1  

VÉASE TAMBIÉN

fstat

(2), isatty(3)
