3.0.0.- Acceso

>>Qué software básico necesito para conectar a Internet?

UN SISTEMA OPERATIVO, por ej. Windows95, NT, OS/2 o Linux. Un Stack de TCP/IP, por ej. Trumpet (Win16/32), Acceso Remoto a Redes (Windows95 y NT), IAK-IBM TCP/IP-WarpConnect (OS/2) y soporte de red (Linux). Ademas de programas de internet varios para cada servicio a usar.

3.1.0.- DOS

Sistema operativo ideal si no tienes una maquina muy potente, hay software para conseguir conectar a Internet incluso con un 286

>> Qué programas necesito para conectar desde MSDOS a Internet ?

3.1.1.- Doslinx

ftp://ftp2.cc.ukans.edu/pub/WWW/Doslynx/
Podría ser un buen navegador Internet para DOS, pero hace mucho tiempo que no actualizan la versión.

3.1.2.- Minuet

ftp://minuet.micro.umn.edu/pub/minuet/
Minuet es un buen navegador Internet para DOS, aunque hace tiempo que no actualizan la versión y tiene algunos bugs y cuelgues frecuentes

3.1.3.- Nettamer

    http://people.delphi.com/davidcolston/
Nettamer es un navegador Internet para DOS (bastante eficiente) y del que están sacando nuevas versiones mejoradas cada tres meses. Aunque el navegador es un poco "feillo", tanto el FTP como el E-mail y News son muy eficientes y rápidos de ejecución y, lo que todavia es mucho mejor NO SE CUELGA NUNCA :) Dispone de versiones para 286 (sin posibilidad de gráficos), 386 en modo gráfico y palmtop. Como ventaja importante, te permite realizar de forma automática la conexión a la hora que quieras con tu servidor, para entregar y recoger el correo y las news, desconectando despues el solo. Te permite realizar sin problemas WWW, FTP, Telnet, Mail, News, etc. Y tiene una ventaja, todo el proceso de conexión y establecimiento de la sesion PPP lo realiza el mismo programa, no como los otros que debes tener ademas el programa PPP aparte. En este site conectamos directamente con el fabricante del soft Nettamer, disponiendo de las versiones para test.
Otros sites para bajarse las versiones oficiales son :
  • ftp://ftp.simtel.net/msdos/internet/
  • ftp://garbo.uwasa.fi/pc/connect/
  • ftp://oak.oakland.edu/pub/simtelnet/msdos/internet/
  • ftp://wuarchive.wustl.edu/pub/MSDOS_UPLOADS/net/tamer/

3.1.4.- Arachne

    http://www.naf.cz/arachne/
Arachne 1.07. Navegador gráfico WWW para DOS. Pocos requerimientos de hardware (8086, 640KB). Soporta HTML 3.2. Incluye PPP dialer, SMTP/POP3.
