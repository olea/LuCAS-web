5.1.0.- NAVEGADORES

5.1.1.- Netscape (Win/Mac/Linux)

El navegador mas popular usado actualmente y disponible practicamente para todas las plataformas, es considerado actualmente más seguro que su eterno rival, el Internet Explorer de Microsoft. Tiene además gestor de e-mail y news y en las versiones Gold incorpora un sencillo editor HTML
 
http://www.netscape.com 
http://home.es.netscape.com/es/ 

5.1.1.1.- Netscape para OS/2

http://www.software.ibm.com/os/warp/netscape 

5.1.2.- I. Explorer (Win)

El navegador de Microsoft también llamado por sus detractores el "Internet exploiter", por los numerosos bugs aparecidos en su lanzamiento, incorpora también gestor de email y news.
 
http://www.microsoft.com/ie/default.htm  

5.1.3.- Opera (Win)

Navegador que soporta todas las opcciones de Netscape e Iexplorer pero que consume muchos menos recursos que estos.
 
http://traviata.nta.no/opera.htm 

5.1.4.- Arachne (DOS)

Arachne es un navegador gráfico para computadoras que utilizan un sistema operativo compatible DOS. Requiere una CPU compatible i8086, 640 kB de memoria y una tarjeta graficadora VGA o SVGA. Sin imágenes, funciona incluso con tarjetas VGA o EGA Arachne trae soporte para HTML 3.2. Desarrollado por xChaos software, el archivo incluye una utilidad de marcación PPP (dialer), visores MPEG y WAV, clientes telnet, ftp y pop 3, etc, directamente integrados a Arachne.
    http://www.naf.cz/arachne/

5.1.5.- Navegadores Off-line

Basicamente hay dos formas de navegar por la www sin estar conectado: una es utilizando los ficheros almacenados en la caché de nuestro navegador y la otra consiste en bajarse un site completo para poderlo consultar tranquilamente off-line. Para facilitar estos dos metodos existen algunas utilidades como:

5.1.5.1.- MSIE Cache Explorer (Win)

    http://www.mwso.com/
Utilidad que sirve para navegar por el CACHÉ que tengas en tu HD (Solo para Internet Explorer)

5.1.5.2.- Netscape Cache Explorer (Win)

    http://www.mwso.com/
Utilidad que sirve para navegar por el CACHÉ que tengas en tu HD (Solo para Netscape)

5.1.5.3.- Teleport Pro (Win)

    http://www.tenmax.com
Teleport permite la visualización de páginas Web sin necesidad de estar conectado. Teleport te graba en el disco duro las páginas que desees de una dirección que le des, y con una profundidad de niveles que le indiques. Una vez configurada la dirección a obtener, el programa busca las páginas que cumplan las condiciones que le hayas indicado y las graba en tu disco duro. Teleport (a diferencia de otros navegadores off-line) graba los ficheros de un site _tal_cual_ de esta forma puedes visualizarlos off-line con cualquier navegador. Nota: Limitado a 200 archivos por proyecto en la versión sin registrar.

5.1.5.4.- WebHacker (Win)

    http://www.ffg.com/whacker/
WebHacker funciona de forma muy similar a Teleport con la diferencia de que graba los sites en un formato propio con lo que hay que verlos off-line
partiendo de éste.

5.1.5.5.- Secret Agent (Win)

    http://www.ffg.com/whacker/
Más reducido que el Teleport pero es muy sencillo de manejar. Graba las páginas en un formato propio pero permite hacer volcados a formato Html de forma que puedes guardar las páginas de esta forma para poder verlas con browsers normales. Lo mejor de este programa es el análisis de la caché del navegador. La versión shareware es operativa sólo 30 días.

5.1.5.6.- About My Cache (Win)

    http://wwwusers.imaginet.fr/~fcrevola/about/

5.1.5.7.- wwwoffle (Linux)

    http://www.gedanken.demon.co.uk/wwwoffle/ 
Excelente software para Linux que permite bajar sitios web completos y además es un estupendo proxie de http para nuestra máquina o red interna.
