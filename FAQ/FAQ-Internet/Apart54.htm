    5.4.4.- OpenChat/2 (OS/2)

    IRC para OS/2. Compatible con IRCII, así que se puede usar el LiCeScRiPt con él. Es shareware. Se puede conseguir en Tucows, Hobbes, LEO http://www.leo.org , etc...

    Sólo comento de él que la lista de servidores IRC está en un fichero ASCII llamado IRCSERV.TXT, y que hay que editar para añadir nuevos servidores IRC.

    5.4.5.- Gammatech IRC (OS/2)

    Sin duda, el mejor programa de IRC para OS/2. Es comercial (de SoftTouch Utilities). Se puede conseguir una demo de la versión 2.05 en Tucows http://tucows.arrakis.es o en su propio site http://www.softtouch.com

    Se pueden usar scripts escritos en el lenguaje de programación REXX.

    5.4.6.- IRCII (OS/2)

    Es idéntico al cliente IRC del mismo nombre disponible para Unix, pero en versión nativa OS/2. La última versión disponible es la 3.0, y no creo que se desarrollen nuevas versiones, ya que el autor está desarrollando un nuevo cliente IRC (Openchat/2). Es modo texto. Se puede conseguir en Hobbes, LEO, Tucows, Shareware.com, etc...

    5.4.7.- IRC/2 (OS/2)

    Es otro cliente IRC en modo texto para OS/2, que no debemos confundir con el IRCII. Es compatible con el IRCII, y soporta la mayoría de sus scripts (aunque no el 100%). Incluye algunos comandos nuevos, que sólo existen en este cliente IRC. Sus autores están desarrollando también una versión Presentation Manager (PM), que tendrá mayores prestaciones que la versión en modo texto. Es shareware y se puede conseguir en http://www.invlogic.com

    5.4.8.- BitchX (Linux, Win, OS/2, Mac)

    El cliente IRC en modo texto mas usado en Internet. Es multiplataforma debido a la disponibilidad del codigo fuente, aunque mayormente es usado en plataformas Unix/Linux. En realidad es una versión modificada del IRCII, por lo que funcionan todos los comandos de éste. Se puede conseguir el codigo fuente y versiones ya compiladas en http://www.BitchX.com  Personalmente recomiendo bajar el codigo fuente y compilarlo uno mismo para poder usar el juego de carácteres Latin1 y poder disfrutar de acentos y eÑes, tan sólo hay que modificar una linea en el fichero config.h y compilarlo, ver documentación para más detalles }:)

    5.4.8.1.- Scripts para BitchX

    Prácticamente todos los scripts para IRCII funcionan con BitchX, aunque el autor recomienda usar scripts que se actualizen periodicamente pues el BitchX está en continuo desarrollo, personalmente he usado ENiGMA, Coherent y ahora (C)rackrock; puedes encontrar estos y algunos más en http://scripts.BitchX.com
