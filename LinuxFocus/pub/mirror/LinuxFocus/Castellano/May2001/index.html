¿ Tu abuela no quiere cambiar a Linux ?

Aún hay muchos usuarios de Windows que dudan sobre instalar Linux. Una de las razones podría ser que, tras 5 reinstalaciones completas de Windows han decidido no tocarlo nunca más y ahora, con que funcione la mayor parte del tiempo son felices. Esto sucede muy a menudo incluso aunque muchos de ellos estarían interesados en ver qué parece.

SuSE ha encontrado la solución a este problema, y se llama CD de evaluación de SuSE en vivo. Es una distribución completa, preparada para ser instalada en un único CD. El CD es botable, y es tambien posible generar primero un floppy de botado si el sistema no es capaz de arrancar desde el CD-ROM.
Este CD está disponible desde SuSE 7.0 y la versión mas reciente es 'live-eval-7.1' para SuSE 7.1

Al botar desde este CD, empezamos con Yast2, la pantalla típica de instalación de SuSE. Que no es una instalación real, sino sólo una configuración. Durante este proceso se crean 3 ficheros en la primera partición de tu primer disco IDE. Da igual que se trate de una partición ext2 de Linux o una partición de Windows, pues el CD en vivo puede manejar ambas.

  • suselive.710, unos 2Mb - con información del CD de evaluación de SuSE 7.1
  • suselive.swp, unos 80Mb - el fichero de swap
  • suselive.usr, unos 100Mb - que es un filesystem ext2 con el directorio home del usuario
Usando ese CD en vivo se puede probar un sistema Linux completo sin ningún riesgo. Ni reparticionar ni recolocar datos. Simplemente crea estos tres ficheros y no deja ninguna otra traza en tu disco duro.

Esta es una forma conveniente de testear una nueva distribución de Linux incluso si ya lo tienes instalado. Pruebas y decides si debes actualizar o si tu distribución actual es suficientemente buena. ¿Desventajas? Que corre desde un CD-ROM y eso quiere decir que es más lento que un sistema Linux normal, puesto que la lectura de datos desde un CD es un proceso relativamente lento.

Puedes descargar la imagen ISO desde ftp://ftp.suse.com/pub/suse/i386/live-eval-7.1/, o desde cualquiera de sus mirror. El sitio principal de SuSE se encuentra a menudo bastante cargado. Si no dispones de un modem DSL para manejar una descarga de 650Mb en un tiempo razonable, también se puede pedir a SuSE, o tal vez pedirle una copia a un amigo. Tal vez haya revistas impresas en tu país que incluyan el CD de evaluación en vivo en alguno de sus números.

Es una buena manera de promocionar Linux entre tus amigos y tal vez, al final, no sólo tu abuela quedará convencida de que se trata de un sistema mejor, y cambiará a Linux.

La pequeña ayuda de LinuxFocus

¿Cómo imprimir varias páginas en cada hoja de papel? Intenta esto. Haz click en el botón de impresión de netscape (o cualquier otro programa) y reemplaza el comando de impresión con: 

psnup -2 | lpr
Esto te sacará dos páginas por cada página de papel

psnup forma parte de un conjunto de comandos para imprimir y manipular postscript, llamado psutils (disponible en http://www.dcs.ed.ac.uk/home/ajcd/psutils/). Un comando muy interesante es psbook, que reorganiza las páginas de forma que se puede conseguir un librito en tamaño A5 si imprimes 2 páginas en cada hoja A4. Todo lo que te hace falta es doblar el papel por la mitad y graparlo. Claro que necesitas una impresora de doble cara (duplex). El comando de impresión sería 

psbook | psnup -2 | lpr -P duplex
En este ejemplo nuestra impresora duplex se llama duplex. Incluso si no dispones de una impresora de este tipo, psutils siguen siendo muy útil. Este comando te convierte páginas A4 en tamaño US-letter: 
psresize -PA4 -pletter in.ps out.ps
