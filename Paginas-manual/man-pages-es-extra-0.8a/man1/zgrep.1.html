NOMBRE

zgrep - buscar una expresión regular en ficheros posiblemente comprimidos  

SINOPSIS

zgrep

[ grep_opciones ] [ -e ] patrón fichero...  

DESCRIPCIÓN

Zgrep

es usado para invocar grep sobre ficheros comprimidos o "gzipeados". Todas las opciones especificadas se pasan directamente a grep. Si no se especifica ningún fichero, entonces la entrada estándar es descomprimida si fuera necesario, y pasada a grep. De otro modo los ficheros dados son descomprimidos si es necesario y pasados a grep.

Si zgrep es invocado como zegrep o zfgrep entonces egrep o fgrep es usado en vez de grep. Si la variable de entorno GREP está definida, zgrep la usa como el programa grep para ser invocado. Por ejemplo:


    para sh:  GREP=fgrep  zgrep cadena ficheros
    para csh: (setenv GREP fgrep; zgrep cadena ficheros)
