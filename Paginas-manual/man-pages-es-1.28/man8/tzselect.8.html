NOMBRE

tzselect - selecciona un huso horario  

SINOPSIS

tzselect 

DESCRIPCIÓN

El programa tzselect pide al usuario información sobre la localización actual y muestra en la salida estándar la descripción de huso horario resultante. La salida es adecuada como valor para la variable de entorno TZ.

Toda la interacción con el usuario se realiza mediante la entrada estándar y la salida de error estándar.  

VARIABLES DE ENTORNO

AWK
El nombre de un programa awk conforme a POSIX (por defecto: awk).
TZDIR
El nombre de un directorio que contiene los ficheros de datos de los husos horarios (por defecto: /usr/local/etc/zoneinfo).
 

FICHEROS

TZDIR/iso3166.tab
Tabla con los códigos de país de 2 letras y los nombres de los países según el estándar ISO 3166.
TZDIR/zone.tab
Tabla con los códigos de los países, latitudes y longitudes, valores TZ y comentarios descriptivos.
TZDIR/TZ
Fichero de datos de huso horario para el huso horario TZ.
 

ESTADO DE SALIDA

El estado de salida es cero si se obtuvo con éxito del usuario un huso horario y un valor no cero en caso contrario.
