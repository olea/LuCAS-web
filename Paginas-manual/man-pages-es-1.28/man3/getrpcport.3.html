NOMBRE

getrpcport - devuelve un número de puerto RPC  

SINOPSIS

int getrpcport(char *host, int prognum, int versnum, int proto);
 

DESCRIPCIÓN

getrpcport()

devuelve el número de puerto para la versión versnum del programa RPC prognum que se está ejecutando en host y que está usando el protocolo proto. Devuelve 0 si no puede contactar con el "portmapper" o si prognum no está registrado. En el caso en que prognum está registrado pero no con la versión versnum, devolverá un número de puerto (para alguna versión del programa) indicando que el programa está, de hecho, registrado. La discrepancia de versiones será detectada en la primera llamada al servicio.
