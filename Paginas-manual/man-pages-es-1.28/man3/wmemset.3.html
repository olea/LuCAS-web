NOMBRE

wmemset - rellena un array de caracteres anchos con un carácter ancho constate.  

SINOPSIS

#include <wchar.h>

wchar_t *wmemset (wchar_t *wcs, wchar_t wc, size_t n);
 

DESCRIPCIÓN

La función wmemset es equivalente a la función memset, pero para caracteres anchos. Rellena el array de n caracteres anchos que comienza en wcs con n copias del carácter ancho wc.  

VALOR DEVUELTO

wmemset

devuelve wcs.  

CONFORME A

ISO/ANSI C, UNIX98  

VÉASE TAMBIÉN
