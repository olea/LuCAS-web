NOMBRE

MB_LEN_MAX - longitud multibyte máxima de un carácter para todas las localizaciones  

SINOPSIS

#include <limits.h>
 

DESCRIPCIÓN

La macro MB_LEN_MAX es el límite superior del número de bytes necesarios para representar un único carácter ancho para todas las localizaciones.  

VALOR DEVUELTO

Una constante entera >= 1.  

CONFORME CON

ANSI C, POSIX.1  

OBSERVACIONES

Las entidades MB_LEN_MAX y sizeof(wchar_t) no guardan relación alguna entre sí. En la libc GNU, MB_LEN_MAX suele tener un valor de 6 mientras que el valor de sizeof(wchar_t) es 4.
