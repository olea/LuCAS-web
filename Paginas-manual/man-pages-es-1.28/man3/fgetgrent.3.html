NOMBRE

fgetgrent - obtiene una entrada del fichero de grupos  

SINOPSIS

#include <grp.h>
#include <stdio.h>
#include <sys/types.h>

struct group *fgetgrent(FILE *flujo);
 

DESCRIPCIÓN

La función fgetgrent() devuelve un puntero a una estructura que contiene la información de grupos que está en el fichero correspondiente a flujo. La primera vez que se la llama devuelve la primera entrada; a partir de ahí, devuelve las entradas sucesivas. El fichero correspondiente a flujo debe tener el mismo formato que /etc/group.

La estructura group se define en <grp.h> como sigue:

struct group {
        char    *gr_name;        /* nombre del grupo */
        char    *gr_passwd;      /* contraseña del grupo */
        gid_t   gr_gid;          /* identificador del grupo */
        char    **gr_mem;        /* miembros del grupo */
};
 

VALOR DEVUELTO

La función fgetgrent() devuelve la dirección de la estructura con información sobre los grupos, o NULL si no hay más entradas o ha ocurrido un error.  

ERRORES

ENOMEM
No hay bastante memoria para alojar la estructura de información de grupos.
 

CONFORME A

SVID 3
